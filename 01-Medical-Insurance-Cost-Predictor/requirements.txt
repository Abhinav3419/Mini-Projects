# WHY PIN VERSIONS? (Interview concept: "Reproducibility")
# ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
# Without pinned versions:
#   pip install tensorflow    → installs latest (maybe 2.22 tomorrow)
#   Your model was trained on 2.21 → predictions might differ!
#
# "It works on my machine" is the #1 deployment bug.
# Pinned versions ensure: your laptop = Docker = AWS = same results.
#
# Rule: pin MAJOR.MINOR, allow PATCH updates (e.g., >=2.21,<2.22)
